# Code of Conduct

This project uses a simple contributor standard until it is accepted by an umbrella foundation. Contributors must be respectful, inclusive, and constructive. Harassment, intimidation, discrimination, and abusive behavior are not acceptable.

Project maintainers may remove content, close issues, or restrict participation when behavior harms the community.

If accepted as an OWASP project, this file should be replaced or aligned with the current OWASP Foundation Code of Conduct and policies.
